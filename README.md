# Start server

> Begebe dich in den server ordner
> Tippe "npm install" um alle dependencies zu installieren
> führe dann "npm run devStart" aus

# Start client

> Begebe dich in den client ordner
> Tippe "npm install" um alle dependencies zu installieren
> führe dann "npm start" aus
